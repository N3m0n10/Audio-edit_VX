Arquivo de áudio ou microfone ao vivo

Renderiza offline no seu navegador, com a cadeia de efeitos atual aplicada, e baixa um .wav — sem gravação de tela, sem permissões extras. Funciona apenas com um arquivo carregado (não com o microfone ao vivo).

Salva pedaços de ~30s automaticamente enquanto você fala, pra nunca perder a sessão se a aba travar. Grava o sinal original do microfone (sem os efeitos — pra isso, use o modo OBS acima). Exige o microfone ligado; o navegador pode pedir permissão pra baixar vários arquivos seguidos — autorize quando aparecer.

Motor de áudio: Voxis — biblioteca open-source de DSP (Python + JavaScript + C++/WASM), API realtime servida via voxis.org/api. · Os presets acima são cadeias de efeito construídas com o builder effects.* da Voxis, inspiradas nos presets nativos do pacote.

Usar ao vivo numa live (OBS): Fontes → + → Fonte de Navegador → aponte a URL pra este arquivo (hospedado ou file://) → marque "Controlar áudio via OBS" pra rotear o som pelo mixer de áudio do OBS → adicione ?obs=1 no fim da URL pra abrir uma versão sem interface visual (só o áudio roda, útil pra deixar como uma fonte invisível na cena). O áudio continua sendo processado normalmente mesmo com a interface escondida.
